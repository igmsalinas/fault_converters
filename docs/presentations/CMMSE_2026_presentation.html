Context

Why not just use threshold alarms?

02

The gap

Parametric drift migrates poles and zeros — a scalar alarm cannot see it.

H(s) = C(sI − A)−1B + D

Near its operating point a switching converter is a closed-loop LTI system. Component aging perturbs the state matrices (A, B, C, D), reshaping the small-signal transfer function across decades of frequency.

Reading the full Bode response H(jω) exposes capacitor aging, MOSFET on-resistance wear and ESR growth long before any single-point limit is crossed.

Non-linear coupling. A ±10% ESR change reshapes magnitude and phase across decades — not a monotone offset a fixed limit can bound.

High dimensionality. Each health state is a 202-D vector (101 magnitude + 101 phase points); hand-tuned per-point limits do not scale.

Label scarcity. Field faults are rare and unlabeled, so the detector must be trained on healthy data only — self-supervised, one-class.

Edge budget. Inference must run in real time beside the converter, within a few watts and sub-millisecond latency.

Our answer. Contrastive representation learning (CARLA) for sensitivity, TensorRT kernel fusion for μs-scale deployment.

  • Each frequency point carries amplitude (dB) and phase (°).
  • Per-feature z-score standardization (μ, σ fit on train) removes scale disparity between magnitude and phase.
  • Fine 1% steps inside the normal band support precision studies; coarser 5% anomalous steps calibrate thresholds and test the models.
  • Dataset and PSIM configs to be open-sourced on publication for reproducibility.

Architectural inductive biases

Six encoders, one contract.

08
MLP-AE & Conv1D-AE

Dense vs. translation-equivariant.

Local Conv1D kernels (size 5) capture resonance structure at O(N) cost; the MLP is a fully-connected baseline.

LSTM-AE & GRU-AE

Gated recurrence over frequency.

Hidden state propagates along the frequency axis to model roll-off and gradual parametric drift.

VAE & Transformer-AE

Variational and attentional.

Latent z ∼ N(μ, σ2I) with KL regularization; multi-head self-attention for O(N2) global spectral coupling.

Bayesian tuning

Objective = validation AUC.

Keras Tuner searches latent dim, kernel size, activation (GELU/Swish/Mish) and optimizer (AdamW/Lion) under warmup-cosine decay and early stopping.

All six share the encoder–decoder contract z = fenc(x), x̂ = fdec(z), trained by MSE reconstruction and ranked by held-out ROC-AUC. Built on Keras / TensorFlow.

Self-supervised contrastive learning

CARLA separates health from degradation.

11
  • Reconstruction-only autoencoders learn to rebuild normal data — limited discriminative power.
  • Point: spikes, dropouts, band-limited noise.
  • Contextual: amplitude/phase offsets, gain and roll-off slope changes.
  • Collective: resonance shift/addition, bandwidth change, oscillation.
  • Physics-based: ESR growth, inductor saturation, Rds increase — matched to real fault modes.
  • Projection head g(·): a 2-layer MLP maps latents to a 64-D unit sphere where NT-Xent (τ = 0.1) operates.
  • Inference scores non-parametrically by k-NN distance (k = 5) over the projection space — threshold-independent ROC-AUC.

NT-Xent objective

Maximally separate healthy and degraded states in latent space.

LNT-Xent = − log exp( sim(zi, zj) / τ ) Σk≠i exp( sim(zi, zk) / τ )

Objectives, scoring & search

From reconstruction error to a decision.

12

Reconstruction objective (MSE)

Lrecon = 1N Σi=1N ‖ xi − x̂i ‖2

Per-sample anomaly score

s(x) = 1L·F Σt,f ( xt,f − x̂t,f )2

CARLA score — k-NN distance in latent space

sCARLA(x) = 1k Σj ∈ Nk(x) ‖ z(x) − zj ‖

VAE objective — negative ELBO

LVAE = Lrecon + β DKL( q(z|x) ‖ N(0, I) )

Threshold calibration (normal errors only)

τ = P95 { s(x) : y = normal } τ = μN + 3 σN   (std method)

Decision rule

ŷ(x) = 𝟙[ s(x) > τ ]

Bayesian hyperparameter search

θ* = argmaxθ ∈ Θ AUC(θ) αEI(θ) = 𝔼[ max( f(θ) − f(θ+), 0 ) ]

Reconstruction models score by MSE; CARLA scores by k-NN distance in the projection space. Every threshold is calibrated on healthy data only — no anomaly labels required at deployment.

Evaluation protocol

Ranking first, then the operating point.

13

Operating-point metrics

Precision = TPTP + FP Recall = TPTP + FN F1 = 2 · Precision · RecallPrecision + Recall Accuracy = TP + TNTP + TN + FP + FN Separability = μanom − μnorm1 + σnorm

TP / FP / TN / FN from the calibrated decision ŷ(x) = 𝟙[ s(x) > τ ]. All computed with scikit-learn on the held-out test set.

Threshold-free ranking

ROC-AUC = ∫01 TPR  d(FPR) PR-AUC = ∫01 Precision  d(Recall)

Threshold calibration — healthy data only

  • Percentile — τ = Pp of normal reconstruction error (p = 90–97).
  • Std — τ = μN + 3σN.
  • F1-optimal — grid search maximizing validation F1.
  • Youden J — maximize TPR − FPR on the ROC curve.

AUCs are threshold-independent, so they expose true ranking even when a fixed operating point fails to transfer — the key diagnostic for the quantization results ahead.

Timing methodology

How the latencies are measured.

18

Measurement protocol

  • Single-sample batch (n = 1) for every backend — identical conditions, so no format is flattered by batch amortization.
  • 15 warm-up iterations discarded — steady-state cuDNN autotune, JIT and warm caches.
  • 150 timed runs with time.perf_counter() wall-clock; report mean ± std, min / max.
  • Per-sample latency = the single-sample runtime — one metric comparable across every backend.
  • Identical inputs, identical machine — the same held-out test tensors on RTX 4070 + Ryzen 7 5800X.

Every backend — from edge TFLite to the batch-1 TensorRT engines — is timed single-sample (n=1) under the same warm-up and timed-run budget, so per-sample latencies are directly comparable. Conclusion & future work

Sensitive, non-invasive, real-time.

22
  • Modular self-supervised framework (CARLA) for non-invasive converter fault diagnosis.
  • Frequency-response signatures are highly sensitive to subtle parametric degradation.
  • TensorRT delivers sub-millisecond inference on constrained edge hardware without accuracy loss.
State space models

Mamba-style linear-complexity sequences, targeting ~8× speedup over Transformers.

Physics-informed nets

Embed Buck-converter ODEs into the contrastive loss to generalize with less synthetic data.

Power & energy

Extend benchmarks to Watts and Joules-per-inference for thermally constrained systems.

KANs

Kolmogorov-Arnold Networks for symbolic extraction and interpretability.

One reproducible loop — physics-based simulation, self-supervised representation learning and hardware-aware compilation — closes the gap between detection accuracy and real-time edge constraints.
